# Luargl

Lua bindings for the [RGL game framework](https://github.com/rxtthin/rgl). <br/>

# Setup

## Dependencies

-	freetype

## Running

1.	Remove the rgl folder
2.	`./update_rgl.sh`
3.	`./run.sh`

# About

The entry point to the luargl library is src/luargl/luargl.c<br/>
The main entry point is src/main.cpp
